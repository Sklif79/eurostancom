Каталог «Оснастка и инструмент» содержит информацию по вспомогательной оснастке для токарных и фрезерных станков с ЧПУ, представлены типовые статические или неприводные держатели инструмента по DIN 69880, приведены различные формы и виды приводных блоков (головок) для токарных станках с ЧПУ, описаны держатели инструмента по DIN 69871 (применяется на фрезерных обрабатывающих центрах с ЧПУ). По заказным номерам Вы можете быстро осуществить заказ необходимой оснастки. Также мы можем предложить инструментальные патроны для всех станков согласно DIN 2080, DIN 69871, JIS B 6339 (MAS 403 BT), ISO 12164-1 (DIN 69893-1) HSK, ISO 26623 (PSK), DIN 228, а также державки VDI для токарных станков согласно ISO 10889 (DIN 69880) и много другое.

Поставка стандартного шлифовального инструмента происходит максимально оперативно, после подтверждения заказа и согласования условий. Однако, нестандартные задачи на поставку специфического инструмента как по составу, так и по форме - безусловно интересны нам. Правильный подбор шлифовальных кругов происходит в непосредственном взаимодействии между нашей компанией и Заказчиком, для этого в каталоге вы найдете бланк заказа, необходимый для заполнения.
