прессовое и формовочное оборудование

Оборудование, предназначенное для изготовления изделий, как правило, без снятия слоя материала, посредством усилия, прилагаемому к заготовке.

Механические прессы

Предназначены для обработки листового металла методом холодной штамповки, высечки, гибки, вытяжки и формовки. Могут быть одинарного действия, двойного действия, для глубокой вытяжки с механическим приводом

Предназначены для обработки листового металла методом холодной штамповки, высечки, гибки, вытяжки и формовки. Могут быть одинарного действия, двойного действия, для глубокой вытяжки с гидравлически приводом

Резьбонакатные станки

Оборудование позволяет накатывать профиль на поверхностях вращения. Применяется для выполнения резьб, профиля червяка, шлицов без снятия слоя материала

Раскрыть

Пружинонавивочные станки

Станки позволяют изготавливать пружины различных типов из проволоки, а также другие сложные изделия методом холодной деформации

Раскрыть

Вальцы

Предназначены для гибки изделий разнообразных профилей. На валковых станках возможно изготовление круглых, квадратных, многогранных труб, С-, V-, корытообразных изделий, а также выполнение отбортовки, подготовки фальцев и другие операции

Трубогибы и профилегибы

Позволяют гнуть трубы для изготовления пространственной конструкции. При гибке методом намотки и проталкивания может использоваться дорн – для устрания деформаций в месте изгиба
